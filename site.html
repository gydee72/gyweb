Pourquoi GitHub est-il devenu si important ?

La plate-forme revendique pas moins de 27 millions d’utilisateurs —, ce qui représente une bonne partie des développeurs informatiques dans le monde. GitHub est aussi bien utilisé par des développeurs indépendants que par de grandes entreprises comme Google, Apple, Facebook ou encore Amazon, qui s’en servent pour bâtir leurs propres produits. Sa facilité d’utilisation a aussi conquis des développeurs amateurs, qui y publient leurs projets. Pour un développeur, exister sur GitHub est devenu incontournable : les recruteurs y scrutent leurs profils et leurs contributions davantage que leur CV. GitHub ferait partie des sites les plus populaires du monde, selon le classement Alexa.

Avec l’hébergeur gratuit de Github..

Pour la plupart des sites, les solutions de blog de type CMS sont largement démesuré. Elles exigent un hébergement particulier (PHP ou autre), des connaissances techniques et surtout une maintenance à long-terme pour des mises à jour de sécurité , plugins et correctifs de bugs à réaliser. Cependant, il est concevable de développer des sites modernes et performants qui n’ont pas besoin de code PHP , et peuvent donc être hébergés n’importe où, et à moindre coût. Le code sera uniquement constitué de HTML, CSS et JavaScript, et s’exécutera directement dans le navigateur du visiteur, ce qui présente de nombreux avantages : simplicité et efficacité, aucune maintenance, aucun risque de hacking, performance et une liberté absolue.
